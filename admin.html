Admin Panel for TOEFL ITP-like Question Sets

Admin Overview

This admin page is designed to upload a complete TOEFL ITP-like practice set to your GitHub repository.

Complete Set Mode: You must provide three valid JSON files: Listening, Structure, and Reading. After upload, the system will automatically update data/manifest.json.
Security note: Your GitHub token is used only in this browser session to call the GitHub API. Do not share your token publicly.

1. GitHub Repository Access

Use a GitHub token with repository contents read/write permission.

2. New Set Information

This will create files such as listening-set-4.json.

Files that will be created

Fill the set number to preview target file names.

3. AI Prompt Helper

Use this English prompt with ChatGPT or another AI tool to generate compatible JSON.

4. Paste Complete Set JSON

Listening JSON

Expected file: data/listening-set-X.json

Structure JSON

Expected file: data/structure-set-X.json

Reading JSON

Expected file: data/reading-set-X.json

5. Upload to GitHub

After validation succeeds, click upload. The admin will upload the three JSON files and update data/manifest.json automatically.
